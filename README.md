# 🍸 Cocktail Training & Recruitment App

A full-stack web application designed to help users learn cocktail-making skills through interactive quizzes, while also connecting them with employers in the hospitality industry.

## 🚀 Features

- Interactive cocktail training and quiz system  
- User authentication with login functionality  
- Role-based access for users and employers  
- Employer dashboard to track user performance  
- Job listing and application system  
- User progress tracking and performance insights  

## 🛠️ Tech Stack

- **Backend:** PHP  
- **Frontend:** JavaScript, HTML, CSS  
- **Database:** MySQL  
- **Other:** Session-based authentication  

## 🔐 Authentication

- Secure login system with role-based access  
- Separate functionality for standard users and employer accounts  

## 💡 Project Purpose

This project was developed as part of my Honours degree to explore how technology can improve staff retention in the hospitality industry by:

- Helping users develop practical skills  
- Allowing employers to identify and recruit capable candidates  
- Providing a platform that connects training with job opportunities  

## 🌐 Deployment

The application was deployed to a live environment, allowing real-world testing and interaction.

## ⚙️ How to Run Locally

1. Clone the repository  
2. Set up a local server (e.g. XAMPP / MAMP)  
3. Import the database  
4. Configure database connection settings  
5. Run the application via localhost
